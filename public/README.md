Step by step (nggak harus sama)
---

1. inisialisasi repo      --> git init
2. inisialisasi project   --> npm init -y
3. tambah file .env       --> ini untuk inisialisasi variable lingkungan (env variable)
4. tambah file .gitignore --> untuk "abaikan" file yang sesuai pattern-nya
                          --> untuk mencegah data sensitif bocor
5. instalasi dependencies --> npm install express multer @google/genai dotenv
6. Mulai coding
